## 📱 **Interfaces Disponibles**

### **🌐 Aplicación Web**
```
🏠 Dashboard Principal          → Resumen general del sistema
📦 Catálogo de Productos       → Navegación y búsqueda de productos  
👁️ Vista Detallada             → Información completa de cada producto
🛒 Gestión de Carritos         → Administración de compras
⚡ Panel de Administración     → Gestión en tiempo real
```

### **🔌 API REST**
```
GET  /api/products             → Lista productos con filtros
GET  /api/products/:id         → Detalle de producto específico
POST /api/products             → Crear nuevo producto
PUT  /api/products/:id         → Actualizar producto existente
DEL  /api/products/:id         → Eliminar producto

GET  /api/carts/:id            → Ver carrito específico
POST /api/carts                → Crear nuevo carrito
POST /api/carts/:id/product    → Agregar producto al carrito
PUT  /api/carts/:id/product    → Actualizar cantidad de producto
DEL  /api/carts/:id/product    → Remover producto del carrito
```
